# Proyek NLP UTS

## Isi Folder
- `NLP_UTS_Analisis_Sentimen_Indonesia_Colab.ipynb`  
  Notebook utama yang siap dijalankan di Google Colab.
- `data/indonesian_test.csv`  
  Dataset publik yang digunakan, total 500 teks.
- `images/`  
  Folder visualisasi yang dipakai dalam laporan.

## Tema Proyek
Analisis sentimen teks Bahasa Indonesia.

## Metode yang Digunakan
- Text preprocessing:
  - cleaning
  - tokenization
  - stopword removal
  - stemming dengan Sastrawi
- Representasi teks:
  - Bag-of-Words
  - TF-IDF
- Model klasifikasi:
  - Logistic Regression

## Hasil Ringkas
- Bag-of-Words: accuracy 0.75, macro F1 0.685
- TF-IDF: accuracy 0.74, macro F1 0.599
